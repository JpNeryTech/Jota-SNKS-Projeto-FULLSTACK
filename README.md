# Jota SNKS

O **Jota SNKS** é um projeto de estudo que simula uma loja de tênis online.  
Ele faz parte da minha jornada de aprendizado em desenvolvimento, onde utilizo o projeto como um espaço para **praticar, experimentar e evoluir minhas habilidades na área de tecnologia**.

Meu objetivo é me tornar um **desenvolvedor Full Stack**, e este projeto representa uma parte importante dessa caminhada, permitindo que eu desenvolva tanto o **frontend** quanto o **backend** de uma aplicação real.

Até o momento, **já construí a base da interface utilizando HTML e CSS**, criando a estrutura e o layout inicial do projeto. A ideia é continuar evoluindo o sistema gradualmente, adicionando novas tecnologias e funcionalidades até transformá-lo em uma **aplicação Full Stack completa**.

## Tecnologias

### Frontend
- HTML  
- CSS  
- React *(planejado para o futuro)*

### Backend
- Java  
- Quarkus *(API que será desenvolvida futuramente)*

## Propósito do projeto

Mais do que apenas um exercício, o **Jota SNKS** é um projeto criado para acompanhar minha evolução como desenvolvedor, colocando em prática conceitos de **desenvolvimento web, organização de código e versionamento com Git**.
